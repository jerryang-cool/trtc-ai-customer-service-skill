# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05

### Added
- Project scaffold generator (`scripts/scaffold.py`)
- Flask backend with 5 unified action handlers (`POST /action`)
- Frontend with TRTC Web SDK v5 integration
- Trilingual i18n support (Chinese / Cantonese / English)
- Keyword-triggered farewell and human agent transfer
- Optional digital avatar with graceful degradation
- 4-dimension service rating system
- Interactive `start.sh` with guided credential setup
- Reference documentation: architecture, configuration, frontend integration
- Multi-platform skill support: CodeBuddy, Cursor, Claude Code, Codex CLI
